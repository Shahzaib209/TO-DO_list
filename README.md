# Simple To-Do List

A simple, elegant, and responsive To-Do List application built with fundamental web technologies.

## Features

- **Add Tasks**: Quickly add new tasks to your list.
- **Mark Complete**: Click on any task text to mark it as completed (crosses it out).
- **Delete Tasks**: Remove tasks you no longer need.
- **Responsive Design**: Looks great on both desktop and mobile screens.
- **Modern Aesthetics**: Features a beautiful gradient background and glassmorphism UI card.

## Technologies Used

- **HTML5**: For the structure of the application.
- **CSS3**: For rich styling, micro-animations, and responsive layout.
- **Vanilla JavaScript**: For fundamental DOM manipulation and event handling without any external libraries or modern ES6+ syntaxes.

## Getting Started

To run this project locally, simply clone the repository and open the `index.html` file in your preferred web browser.

```bash
git clone <your-repository-url>
cd to_do_list
```
Then, double-click on `index.html` or drag and drop it into a browser window.

## Customization

You can easily customize the look and feel by modifying the `style.css` file. The logic for managing tasks is handled entirely in `script.js`.
